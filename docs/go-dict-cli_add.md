## go-dict-cli add

Add record to dictionary

### Synopsis

Add new record consisting of the word, definition and optionally a tag to your existing dictionary.

Example:
  add <dict-name> -w <word> -d <definition> -t <tag>

```
go-dict-cli add [flags]
```

### Options

```
      --d string   Definition of the word
  -h, --help       help for add
      --t string   Tag for the record (optional)
      --w string   Word to be added to the dictionary
```
