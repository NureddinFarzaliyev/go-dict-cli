## go-dict-cli list

List dictionaries or the content of a dictionary

### Synopsis

List all of the previously created dictionaries or list the content of a specific dictionary.

Example:
	List all dictionaries: dict list 
	List the content of a dictionary: dict list <dict-name>

```
go-dict-cli list [flags]
```
