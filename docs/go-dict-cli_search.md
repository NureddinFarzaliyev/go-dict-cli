## go-dict-cli search

Search for a word or a tag in dictionary.

### Synopsis

Search for a word or for a tag in specific dictionary.

Example:
	Word search: dict search <dict-name> <word>
	Tag search: dict search <dict-name> <word> -t

```
go-dict-cli search [flags]
```

### Options

```
  -h, --help   help for search
  -t, --t      Use this flag if you want to search based on tag instead of word
```
